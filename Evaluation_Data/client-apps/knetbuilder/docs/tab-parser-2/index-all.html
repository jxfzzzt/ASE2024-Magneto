Accession - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
This maps to a column via the ColumnOrValue structure (see). 
Attribute - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
This maps to a column via the ColumnOrValue structure (see). 
Column - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
Elements of type Column might have either the index attribute (pointing to a file column via a 0-n integer), or the header attribute (reporting the header to be found in the file). 
ColumnOrValue - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
A type having a group of this type allows for the definition of a value or of a column reference (see the Column XSD type). 
Concept - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
Defines how a tabular file is mapped to an ONDEX concept. 
Name - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
This maps to a column via the ColumnOrValue structure (see). 
parser - element of namespace http://www.ondex.org/xml/schema/tab_parser
This is the root element, see the Parser XML type. 
Parser - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
The root element <parser>, which contains a few parser-directives (e.g., delimiter) and then child elements that map the tabular file type to parse to ONDEX entities (concept and relations).  
Relation - complexType of namespace http://www.ondex.org/xml/schema/tab_parser
Defines how a tabular file is mapped to an ONDEX relation.
